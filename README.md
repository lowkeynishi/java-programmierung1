# Java-Programmierung 1

This repository contains my Java exercises from the **Java-Programmierung 1** course at Technische Hochschule Brandenburg. It documents my progress in learning core Java concepts through weekly university exercises.

## Repository structure

The exercises are organized by week (`Ubung`) as separate folders:

- `Ubung2` – Exercises 1–4 (basic Java syntax, control structures)
- `Ubung3` – Exercises 1–4 (methods, loops, simple algorithms)
- `Ubung4` – Exercises 1–4 (arrays, input handling)
- `Ubung5` – Exercises 19–21 (Euclidean algorithm, prime number computation)
- `out/production/Ubung5` – Compiled output generated by the IDE

> Note: Folder names and numbers follow the original university exercise sheets.

## Topics covered

- Java basics: variables, data types, operators
- Conditional statements and loops
- Methods and parameter passing
- Arrays and simple data structures
- Mathematical algorithms (e.g. Euclidean algorithm, prime checking)
- Basic console input and output

## How to run

1. Clone the repository:
   ```bash
   git clone https://github.com/lowkeynishi/Java-Programmierung1.git
   cd Java-Programmierung1
   ```

2. Open the project in your IDE (IntelliJ IDEA recommended).

3. Navigate to the desired `Ubung` folder and run the corresponding `main` class.
   - Alternatively, compile and run from the terminal:
     ```bash
     javac Path/To/FileName.java
     java Path.To.FileName
     ```
